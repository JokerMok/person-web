Codex Vibe Coding Tool

公众号文章排版工具

用 Codex Vibe Coding 做出的本地网站小工具，用来把自媒体文章快速整理成适合公众号发布的版式。

Local Web ToolCodex Vibe CodingWeChat Article
项目状态

已具备本地 Web 工程结构，围绕文章解析、样式模板和富文本复制能力持续优化。

Project Context

为什么做

公众号文章从写完到发布，中间有大量格式处理、样式调整和复制粘贴工作，适合做成本地工具。

Problem

排版重复

标题、分隔、引用、列表、强调和段落样式如果手动处理，会消耗大量内容发布时间。

Solution

文章格式自动化

通过解析文章结构、套用样式模板、生成可复制富文本，缩短发布前处理流程。

Role

Codex Vibe Coding

用 Codex 辅助快速搭建本地 Next.js 工具，并持续迭代解析和渲染能力。

Workflow

工具能力

它服务于臻一的内容生产线，是写作到发布之间的格式处理层。

01

输入文章

把草稿文章粘贴进本地工具。

02

结构解析

识别标题、段落、列表、引用和强调内容。

03

样式套用

套用适合公众号阅读的版式模板。

04

复制发布

生成可复制到公众号后台的富文本内容。
