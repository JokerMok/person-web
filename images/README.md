## 替换说明

1. 将您的图片资源按照上述文件名命名
2. 替换到当前 `src/images/` 目录下
3. 确保图片格式为 JPG 格式
4. 建议保持推荐的尺寸比例，以确保最佳显示效果
5. 图片文件大小建议控制在 200KB 以内，以保证页面加载速度

## 技术说明

图片使用了以下CSS类和属性来确保最佳显示效果：
- `w-full h-full`: 确保图片填满容器
- `object-contain`: 保持图片比例并适应容器，避免裁剪
- `transition`: 平滑过渡效果
- `cursor-pointer`: 鼠标悬停时显示指针，提示可交互

### 轮播图技术特点
1. **响应式设计**: 自适应容器尺寸
2. **自动轮播**: 缩略图自动切换，每3秒一张
3. **交互控制**: 支持点击放大、拖动、缩放
4. **模态弹窗**: 点击放大后显示全屏模态框
5. **多端支持**: 适配桌面和移动设备
6. **键盘导航**: 支持ESC键关闭弹窗

### 图片组件
使用 `ImageCarousel` 组件处理图片展示，支持以下特性：
- 缩略图轮播
- 点击放大
- 滚轮缩放
- 拖拽移动
- 自适应尺寸
- 全屏模态
