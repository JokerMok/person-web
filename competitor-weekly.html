Competitive Intelligence

竞品周报

为优废环保的竞品情报场景搭建周报系统。原先流程运行在 Coze 平台，现在迁移为本地部署，并由 Codex 贯穿开发、调试和迭代。

Coze → LocalCodex Built竞品情报自动化
项目状态

已形成周报输出、版式规范、字段规则、提示词、信源列表和本地 Web 项目雏形。

Project Context

项目背景

竞品周报的价值不在于“抓信息”，而是把分散信源变成可持续追踪、可复盘、可汇报的情报系统。

Problem

情报分散

竞品动态、政策、市场动作和产品变化散落在不同信源里，人工整理成本高。

Solution

本地自动化周报

从平台流程迁移到本地部署，保留提示词、字段规则和视觉版式，增强可控性。

Role

Codex 贯穿搭建

用 Codex 协助完成本地工程化、脚本整理、页面项目和流程调试。

Workflow

系统结构

这个项目更接近“企业情报产品原型”，不是一次性报告。

01

信源与字段

定义需要监控的竞品、事件类型、字段和可信度判断。

02

提示词规则

把提取、校验、分类和摘要规则沉淀为可复用知识库。

03

周报生成

输出长图、PDF、Markdown 等形式，支撑内部阅读和汇报。

04

本地部署

从 Coze 迁移到本地 Web/脚本环境，增强维护与迭代能力。
