Codex Vibe Coding Tool

宫格图片切割工具

用 Codex Vibe Coding 做出的本地网站小工具，用于把图片切成适合社媒发布的宫格图，服务小红书、视频封面和内容素材制作。

Local Web ToolCodex Vibe CodingImage Grid
项目状态

已形成本地工具原型，重点解决自媒体视觉素材的快速切割和导出。

Project Context

为什么做

社媒内容生产里，图片经常需要拆成宫格、封面或连续展示素材。手工切图容易慢，也容易尺寸不统一。

Problem

图片处理碎片化

小红书、视频封面、内容素材都需要不同裁切方式，手工重复操作成本高。

Solution

本地切割工具

把图片上传、宫格切割、预览和导出集中到一个本地页面里。

Role

Codex Vibe Coding

用 Codex 快速完成工具原型，把一个明确的小需求做成本地可用网站。

Workflow

工具链路

它是内容生产工作台的一块小组件，追求的是高频、轻量、马上可用。

01

上传图片

导入需要切割的原始图片。

02

选择宫格

选择 4 宫格、9 宫格或适合发布平台的拆分方式。

03

预览切片

确认每一格的画面位置和整体连续性。

04

导出素材

生成可直接用于社媒发布的图片素材。
